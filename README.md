# 🍅 Pomodoro Timer

Um Pomodoro Timer moderno e completo desenvolvido com React e TailwindCSS.

## ✨ Funcionalidades

- ⏱️ Timer Pomodoro com ciclos automáticos
- 🎨 Interface minimalista e responsiva  
- 🌙 Modo escuro automático
- ⚙️ Personalização completa de tempos
- 🔔 Notificações sonoras e visuais
- 📊 Histórico e estatísticas detalhadas
- 🏆 Sistema de conquistas gamificado
- 💾 Persistência local de dados
- 🎭 Animações suaves

## 🚀 Como usar

### Desenvolvimento
```bash
npm install
npm run dev
```

### Produção
```bash
npm run build
npm run preview
```

## 🛠️ Tecnologias

- React 18 + Vite
- TailwindCSS + shadcn/ui
- Lucide React (ícones)
- Web Audio API (sons)
- Notification API

## 📱 Responsivo

Funciona perfeitamente em desktop, tablet e mobile.

## 🎯 Técnica Pomodoro

- 25 minutos de foco
- 5 minutos de pausa curta  
- 15 minutos de pausa longa (a cada 4 ciclos)
- Tempos personalizáveis
